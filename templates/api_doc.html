PasteBum API Documentation

PasteBum API allows you to use PasteBum features. You can using only url, create a note or view features of a note. These, come as json data.

Table of Contents

  • Create Note
  • View Note

Create Note

Path: /api/create_note/<title>/<content>/<syntax>

Parameters

title: Title of note.
content: Content of note.
syntax: Syntax of note


Return

Return a url as json data.
Example: [{"url":"f70303d3ed"}]

View Note

Path: /api/view_note/<url>

Parameters

url: Url of note.


Return

Return features of the note as json data.
Example: [{"content":"lorem ipsum dolor sit amet","details":[{"data_size":37,"date":"2020-11-27 17:01:31","line":"1\n"}],"syntax":"plain text","title":"Lorem Ipsum","url":"ba9c2f3e14"}]

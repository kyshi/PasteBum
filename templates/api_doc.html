PasteBum API Documentation

PasteBum API allows you to use PasteBum features. You can create a note or view features of a note. These, come as json data.

Create Note

Create a note.

Path: /api/create_note/<title>/<content>/<syntax>

Parameters

title: Title of note.
content: Content of note.
syntax: Syntax of note.


Return

Return the url of created note as json data.
Example: {"url":"f70303d3ed"}

View Note

View features of a note.

Path: /api/view_note/<url>

Parameters

url: Url of note.


Return

Return features of the note as json data.
Example: {"content":"lorem ipsum dolor sit amet","details":[{"data_size":37,"date":"2020-11-27 17:01:31","line":"1\n"}],"syntax":"plain text","title":"Lorem Ipsum","url":"ba9c2f3e14"}

Errors

Can be errors because of Wrong usage of API. These errors return as json. This json structure like this: {"error": "Cause of error"}

Rate Limit

If you exceeded rate limit, will return this error.

Parameters

error: Too many request.


Return

Return error as a json.
Example: {"error": "Too many request."}

Invalid Url

If you enter not exsist url, will return this error.

Parameters

error: Invalid url.


Return

Return error as a json.
Example: {"error": "Invalid url. "}

Invalid Syntax

If you enter not exsist syntax, will return this error.

Parameters

error: Invalid syntax.


Return

Return error as a json.
Example: {"error": "Invalid syntax. "}
